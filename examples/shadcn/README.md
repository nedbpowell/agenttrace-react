# shadcn Example

Next.js 14 example showing `agenttrace-react` styled with shadcn/ui-style components instead of raw Tailwind markup.

## Run

```bash
corepack pnpm install
corepack pnpm --filter shadcn dev
```

## What it demonstrates

- `agenttrace-react` headless primitives
- `agenttrace-langgraph` mock stream integration
- shadcn-style `Button`, `Card`, `Badge`, and `Separator` primitives
- a replayable approval-in-the-loop demo run
